How to Create a Form

  1. Insert "form" element
    • the "action" element attribute determines where to send the form data when the form is submitted. e.g., you can input a "mailto" attribute
    • the "method" element attribute determines the HTTP method to use when sending form data
    • the "enctype" element attribute specifies how the form data should be encoded when submitting the form. e.g. plain text etc.
  2. Insert "label" element in order to put in the name of the field
  3. Insert "input" element followed by input attribute "type="xxx"",and isnert description of the data in the input attribute "name="xxx"".
  4. There are many different form input attributes that can be found at this link
