How to make hyperlinks

  1. Use anchor html element "a".
  2. Insert element attribute "href" which stands for hyperlink reference.
  3. after the =, insert link to the webpage the link should lead to
  4. insert the text that you want to be hyperlinked after the angled bracket
  5. close the code with "/a"

If the webpage you want to link to is in the same priority as the existing webpage,you only need to key in the name of the webapge as shown in this application.

The default link will be blue in color for links you have never clicked and purple for links you have visited before.
