How to Create a Table

  1. Insert element "table"
  2. Insert elements "thead" for header, "tbody" for body, and "tfoot" for footer
  3. Insert "th" for each table header row. The text will automatically be bolded and centered
  4. Insert "tr" for each table row
  5. There are no coloumns, columns are created by inserting "td", table data, in each row from left to right

*Although it is possible to use "th" element to insert headers in the table body, it is not recommended as it will make it difficult to isolate the header and style it, etc., later.

*The table footer is used to provide summary information and will always appear at bootom of table regardless of where it appears in the HTML unless overridden with CSS.

*You can also create tables within tables, but it is better to use CSS to effect table layout

Attributes for Table element

Attribute Description
border="n" Used to create borders for table.
cellspacing="n" Used to insert space between cells
cellpadding="n" Used to insert spacing between content and border of cell
