How to insert images into webpage

Images hosted on servers
  1. Use HTML element "img"
  2. Insert element attribute "src"
  3. right click on the image that you want to appear on the browser and copy image address
  4. paste image address after =

If you wish to upload your own photo, you can use an application / service provider like photobucket.
If the server is taken down or the image is moved / deleted in the server, the image will no longer show up on your webpage


Images hosted locally on same folder as project
  1. Use HTML element "img"
  2. Insert element attribute "src"
  3. upload photo into project file
  4. insert name of photo or path of photo after =

If the image is in the same hierachical level as the webpage, you only need to key in the name of the image as shown in this application. If not, you will have to add a path for the image by putting the folder name followed by "/".
